REST interface

GET

  • convert?q={ckan-dataset-id} - convert CKAN dataset json to datapackge json e.g. /convert?q=0c03fa08-2142-426b-b1ca-fa852f909aa6
  • convert_resources?q={ckan-dataset-id} - convert CKAN dataset json to datapackge json with resources, also if any of the resources files are csv file the tabular data package will be converted. e.g. /convert_resources?q=grassroots-frictionless-data-test

  • package_json?q={ckan-dataset-id} - download CKAN dataset json to datapage.json file e.g./package_json?q=0c03fa08-2142-426b-b1ca-fa852f909aa6
